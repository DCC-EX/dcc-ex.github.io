************************************
Supported Motorboards Setup Notes
************************************

These pages are notes and guides for how to connect and configure each specific motor controller supported by DCC++ EX. Click on the motor controler you want to use. Be sure to read :doc:`Motor Board Config </advanced-setup/motor-board-config>` as well. For a listing of the different types of motor boards and whether they are supported or not, see the :doc:`Motor Boards </reference/hardware/motor-boards>` page.
