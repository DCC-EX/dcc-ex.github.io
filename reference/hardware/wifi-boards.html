WiFi Boards¶

For more information see Wifi Setup

Makerfabs WiFi Shield¶

  • Makerfabs ESP-8266 WiFi Shield (recommended)

Duinopeak WiFi Shield¶

  • Duinopeak ESP-8266 WiFi Expansion Board (plus an ESP-01 or 01s)

See the ESP-01S below that installs on this board

ESP-01S¶

  • ESP-01 or ESP-01S Board (This is not a shield. You will need to use jumpers)

You can use this board stand-alone with jumpers, or use the Duiopeak ESP-8266 Wifi Expansion Board above and plug this board into it.
